Module synthrs::writer[−][src]

Writes audio samples to files. (Reads too, this module needs a rename)

Audacity PCM import settings

  • File > Import > Raw Data…
  • Signed 16-bit PCM
  • Little-endian
  • 1 Channel (Mono)
  • Sample rate: 44_100Hz (or whatever your samples generated have)
